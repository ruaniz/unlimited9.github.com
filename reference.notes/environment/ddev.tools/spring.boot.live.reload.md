# SpringBoot live reload

#### add dependency : devtools 
$ vi build.gradle
```
...
implementation 'org.springframework.boot:spring-boot-devtools'
...
```

#### add property 
$ vi application.properties
```
...
spring.devtools.livereload.enabled=true
...
```
