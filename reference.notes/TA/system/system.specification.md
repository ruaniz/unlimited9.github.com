## system specification

#### system information
$ dmesg > dmesg.txt

#### CPU info
$ cat /proc/cpuinfo

#### memory
$ cat /proc/meminfo

$ free -m

#### disk
$ df -h

#### network : IP
$ ifconfig

#### hostname, kernel version
$ uname -a  

$ hostname

#### release info
$ cat /etc/issue.net

#### system architecture - bit
$ getconf LONG_BIT
